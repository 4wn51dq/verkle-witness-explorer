###### Tool that simulates Ethereum stateless execution inputs by generating and verifying verkle proofs



using libraries/ repositories:

https://github.com/ethereum/go-verkle (verkle library)
https://github.com/lunfardo314/verkle (for kzg or kate polynomial commitment scheme)
https://github.com/dedis/kyber (for cryptographic implementations)

	•	Verkle tree structure
	•	Polynomial commitments
	•	Proof aggregation
	•	Stateless verification
	•	Witness idea (Ethereum-style)
